## Bean_DailyLog
DALL-E를 이용해 나만의 그림일기를 작성하는 사이트입니다.    
사용자가 원하는 분위기를 선택하면 DALL-E로 그림을 받아 게시글을 작성한 후 다른 사람들과 소통할 수 있습니다.
